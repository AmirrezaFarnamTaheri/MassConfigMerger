About ConfigStream

Our mission is to make the internet more open and accessible for everyone.

Our Mission

ConfigStream is an automated system that finds, tests, and shares free VPN configurations. We believe that everyone deserves secure and unrestricted access to information, and this project is our contribution to that cause.

How It Works

Using GitHub Actions, our system runs around the clock to fetch fresh proxy lists from public sources. Each proxy undergoes rigorous automated testing for speed, reliability, and security. Only the best, fully-vetted proxies are included in our configurations, which are updated every three hours.

Important Notice

These are free, public proxies. While we filter out malicious nodes, we cannot guarantee your privacy. Please avoid using these proxies for sensitive activities like online banking or handling personal data. For casual browsing and bypassing geo-restrictions, they are an excellent free resource. Use at your own risk.

Technology Stack

This project is built with a variety of open-source technologies, including:

  • Python for the core logic and data processing.
  • GitHub Actions for automation and continuous integration.
  • Chart.js for data visualization.
  • Feather Icons for the user interface.
