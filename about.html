About ConfigStream

An open-source project dedicated to internet freedom through automated VPN configuration aggregation.

What is ConfigStream?

ConfigStream is an open-source VPN configuration aggregator that automatically collects, tests, and publishes free VPN configurations from public sources. Our mission is to provide reliable, tested, and accessible proxy configurations to help users bypass censorship and access the open internet.

How It Works

The project runs entirely through GitHub Actions, updating configurations every few hours. Each configuration is rigorously tested for connectivity, latency, and security before being made available. This ensures you always have access to fresh, working proxies without manual intervention.

Security & Privacy

These are FREE public VPN nodes from unknown operators. While we perform basic security checks, we cannot guarantee complete safety. Never use them for sensitive activities like banking. They're best for casual browsing and bypassing geo-restrictions. Use at your own risk.
