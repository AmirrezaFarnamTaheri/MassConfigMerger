About ConfigStream

An open-source project dedicated to internet freedom through automated VPN configuration aggregation

What is ConfigStream?

ConfigStream is an open-source VPN configuration aggregator that automatically collects, tests, and publishes free VPN configurations from public sources. Our mission is to provide reliable, tested, and accessible proxy configurations to help users bypass censorship and access the open internet.

The project runs entirely through GitHub Actions, updating configurations every 6 hours. Each configuration is rigorously tested for connectivity, latency, and security before being made available. This ensures you always have access to fresh, working proxies without manual intervention.

How It Works

ConfigStream follows an automated pipeline that ensures high-quality, working configurations:

  • Fetch: Collects VPN configurations from multiple public sources and aggregators
  • Parse: Extracts and validates configuration details for all supported protocols
  • Test: Uses sing-box to test connectivity and measure latency for each config
  • Security Check: Runs comprehensive security tests including:
    • Content injection detection
    • Header preservation verification
    • Redirect handling tests
    • SSL/TLS validation
  • Geolocate: Determines country, city, and network provider using GeoIP databases
  • Filter: Removes non-working and insecure proxies from the final output
  • Sort: Orders configurations by performance (latency)
  • Generate: Creates output files in multiple formats (Base64, Clash, Raw)
  • Publish: Commits changes to GitHub and auto-deploys to GitHub Pages

Important Warning

These are FREE public VPN nodes from unknown operators. While we perform comprehensive security checks, we cannot guarantee complete safety.

What We Test:

  • Content Injection: Detects if proxies modify page content or inject malicious scripts
  • Header Preservation: Ensures HTTP headers are not stripped or modified
  • Redirect Handling: Verifies proper HTTP redirect behavior
  • SSL/TLS Security: Tests HTTPS connection security and certificate validation

What We Cannot Guarantee:

  • Traffic logging - Operators may log your activities
  • Data privacy - Your data may be monitored or stored
  • Continuous availability - Free proxies often go offline
  • Bandwidth or speed - Performance can vary significantly
  • Operator intentions - Unknown providers may have malicious intent

Frequently Asked Questions

Are these VPNs safe to use?
They are tested for basic security issues, but since they're from unknown operators, we cannot guarantee complete safety. Never use them for sensitive activities like banking. They're best for casual browsing and bypassing geo-restrictions.
Do I need to update configurations manually?
No! If you use a subscription link, your client will automatically fetch updates when you refresh. We recommend updating your subscriptions every 6-12 hours to get the freshest configs.
Why do some proxies stop working?
Free public proxies are often temporary. They may go offline, change configuration, or become overloaded. That's why we update every 6 hours - to ensure you always have access to working proxies.
Can I contribute my own proxy sources?
Yes! Fork the repository, add your sources to sources.txt, test locally, and submit a pull request. We welcome community contributions!
How can I run this locally?
Clone the repository, install Python 3.11+, install dependencies with pip install -e ., and run configstream merge --sources sources.txt. Full instructions are in the README.
Which client should I use?
Android: V2RayNG, V2Box, or NekoBox
iOS: Shadowrocket, V2Box, or Stash
Windows: Clash for Windows or V2RayN
macOS: ClashX or V2RayX
Linux: Clash or v2ray-core
