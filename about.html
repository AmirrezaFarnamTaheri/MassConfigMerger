What is ConfigStream?

ConfigStream is an open-source project that automatically aggregates, tests, and provides free VPN and proxy configurations. Our system runs every three hours, gathering configs from public sources, testing them for connectivity and performance, and making the best ones available for easy download.

Security & Privacy

Use with caution. These are public proxies from unknown operators. While we perform basic security checks (like filtering nodes that inject ads or malicious scripts), we cannot guarantee their safety. They are best suited for bypassing geo-restrictions, not for handling sensitive data. Always prioritize using HTTPS.

Technology Stack

This project is powered by Python and a suite of open-source tools:

  • Python: The core logic for fetching and processing configs.
  • sing-box: For testing proxy connectivity and latency.
  • GitHub Actions: For automating the update process every 3 hours.
  • Chart.js: For rendering the beautiful charts on the statistics page.
  • Feather Icons: For the clean and simple icons used throughout the site.
