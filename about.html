What is ConfigStream?

ConfigStream is an automated VPN configuration aggregator that collects, tests, and publishes free VPN configurations from public sources. The entire process is automated through GitHub Actions and runs every 6 hours to ensure fresh, working configurations are always available.

How It Works

ConfigStream follows a simple but effective pipeline:

  1. Fetch: Collects VPN configurations from multiple public sources
  2. Parse: Extracts and validates configuration details
  3. Test: Tests each configuration for:
    • Connectivity (can it connect?)
    • Latency (how fast is it?)
    • Security (is it safe?)
  4. Geolocate: Determines country, city, and network provider
  5. Filter: Removes non-working and insecure proxies
  6. Sort: Orders by performance (latency)
  7. Generate: Creates output files in multiple formats
  8. Publish: Commits to GitHub and deploys to GitHub Pages

Security Tests

ConfigStream performs several security checks on each proxy:

  • Redirect Handling: Ensures proxies handle HTTP redirects properly
  • Header Preservation: Verifies that HTTP headers are not stripped
  • Content Injection: Detects if the proxy modifies page content
  • SSL/TLS Check: Tests the security of HTTPS connections

Proxies that fail these tests are marked as insecure and not included in the final output.

Supported Protocols

ConfigStream supports a wide range of VPN protocols:

  • VMess - Popular V2Ray protocol
  • VLESS - Lightweight V2Ray protocol (including REALITY)
  • Shadowsocks - Fast and secure proxy
  • Trojan - TLS-based proxy protocol
  • Hysteria / Hysteria2 - UDP-based high-performance protocol
  • TUIC - QUIC-based proxy protocol
  • WireGuard - Modern, fast VPN protocol
  • Naive - Censorship-resistant proxy
  • HTTP/HTTPS/SOCKS - Traditional proxy protocols

Output Formats

ConfigStream generates several output formats:

1. Base64 Subscription

Universal format compatible with most V2Ray clients (V2RayNG, V2Box, etc.). Simply paste the subscription link into your client.

output/vpn_subscription_base64.txt

2. Clash Configuration

Ready-to-use YAML file for Clash and Clash Meta clients. Download and import directly.

3. Raw Configs

Unencoded configuration links for manual import or advanced users.

4. JSON Data

Detailed proxy information in JSON format, including:

  • Protocol, country, city, ASN
  • Latency and performance metrics
  • Security test results
  • Full configuration strings

Supported Clients

ConfigStream works with most popular VPN clients:

  • V2RayNG (Android) - Use Base64 subscription
  • V2Box / Shadowrocket (iOS) - Use Base64 subscription
  • Clash for Windows - Use Clash YAML file
  • ClashX (macOS) - Use Clash YAML file
  • Clash Meta / Clash Verge - Use Clash YAML file
  • Hiddify / NekoBox / SagerNet - All formats supported
  • Sing-Box - Use raw configs

FAQ

Q: Are these VPNs safe to use?

A: They are tested for security issues, but since they're from unknown operators, we cannot guarantee complete safety. Never use them for sensitive activities like banking.

Q: Do I need to update the configurations manually?

A: No! If you use a subscription link, your client will automatically fetch updates.

Q: Why do some proxies stop working?

A: Free public proxies are often temporary. They may go offline, change configuration, or become overloaded. That's why we update every 6 hours.

Q: Can I contribute my own proxy sources?

A: Yes! Fork the repository, add your sources to sources.txt, and submit a pull request.

Q: How can I run this locally?

A: Clone the repository and follow the installation instructions in the README. You'll need Python 3.8+ and the required dependencies.
