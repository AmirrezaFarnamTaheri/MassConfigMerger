Features

  • Automatically updated every 6 hours via GitHub Actions
  • All configurations tested for connectivity and performance
  • Security checks to filter out malicious nodes
  • Geolocation data for each proxy (country, city, ASN)
  • Multiple output formats (Base64, Clash, Raw)
  • Sorted by latency for optimal performance
  • 100% open source and transparent
  • No registration or sign-up required

How to Use

  • Download your preferred configuration format above
  • For V2RayNG/V2Box: Use the subscription link or Base64 file
  • For Clash clients: Download the Clash YAML file
  • Import into your VPN client
  • Bookmark this page for fresh configs every 6 hours
  • Check the Proxies page for detailed information

Important Security Notice

  • These are FREE public VPN nodes from unknown operators
  • Traffic may be logged, monitored, or modified
  • NOT suitable for banking, sensitive data, or private activities
  • Best for casual browsing and bypassing geo-restrictions
  • Always prefer HTTPS websites when using these proxies
  • We perform security tests, but cannot guarantee 100% safety
  • Use at your own risk - no warranties provided

Supported Clients

  • V2RayNG (Android) - Use Base64 subscription
  • V2Box / Shadowrocket (iOS) - Use Base64 subscription
  • Clash for Windows / ClashX - Use Clash config
  • Clash Meta / Clash Verge - Use Clash config
  • Hiddify / NekoBox / SagerNet - All formats supported
  • Sing-Box - Use raw configs
