Quick Start Guide

This guide will walk you through the basic steps to get ConfigStream up and running.

Step 1: Installation

First, clone the repository and install the required dependencies:

git clone https://github.com/amirrezafarnamtaheri/configStream.git
cd configStream
pip install -e .[dev]

Step 2: Configuration

Copy the example configuration file and edit it to your needs. At a minimum, you must set a `SECRET_KEY` in the `security` section.

cp config.yaml.example config.yaml
# Open config.yaml and add your secret key

Step 3: Run the Daemon

Start the web dashboard and the background scheduler by running the `daemon` command:

python -m src.configstream.main_daemon

You can now access the web dashboard at http://127.0.0.1:8080.
