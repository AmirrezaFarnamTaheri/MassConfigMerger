Quick Start Guide

Get up and running with ConfigStream in minutes.

1. Installation

First, make sure you have Python 3.8 or higher installed. Then, you can install ConfigStream using pip:

pip install configstream

2. Configuration

Create a config.toml file to customize your settings. You can specify your output directory, sources, and other options. Here's a basic example:

[output]
output_dir = "data"

[sources]
# A list of URLs to fetch VPN configurations from.
urls = [
    "https://raw.githubusercontent.com/barry-far/V2ray-Configs/main/All_Configs_Sub.txt",
    "https://raw.githubusercontent.com/yebekhe/V2Hub/main/merged"
]

[telegram]
# Enable fetching from Telegram channels.
enabled = true
channels = [
    "https://t.me/s/v2ray_outline"
]

3. Running the Daemon

The easiest way to get started is by running the daemon, which includes the web dashboard and a scheduler for automatic testing:

configstream daemon

This will start the web server, and you can access the dashboard at http://localhost:8080.

4. Using the Dashboard

The dashboard provides a visual overview of your VPN configurations. You can see which ones are working, their latency, and other useful information. You can also manually trigger tests and manage your settings.
