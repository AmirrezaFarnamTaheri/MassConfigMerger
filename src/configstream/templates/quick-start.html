Quick Start Guide

Get up and running with ConfigStream in just a few simple steps.

1. Installation

First, make sure you have Python 3.8 or higher installed. Then, you can install ConfigStream directly from the GitHub repository:

pip install git+https://github.com/amirrezafarnamtaheri/configStream.git

2. Configuration

Create a config.yaml file in the root of the project to customize your settings. You'll also need a sources.txt file to specify your subscription URLs. Here's a basic example:

config.yaml:

output:
  output_dir: "output"
  write_clash: true
  write_base64: true

processing:
  enable_url_testing: true
  max_ping_ms: 1000

sources.txt:

https://raw.githubusercontent.com/barry-far/V2ray-Configs/main/All_Configs_Sub.txt
https://raw.githubusercontent.com/yebekhe/V2Hub/main/merged

3. Running ConfigStream

The easiest way to get started is by running the daemon, which includes the web dashboard and a scheduler for automatic testing:

configstream daemon

This will start the web server, and you can access the dashboard at http://localhost:8080.

You can also run a full fetch, test, and merge cycle directly from the command line:

configstream full

4. Using the Dashboard

The dashboard provides a visual overview of your proxy configurations. You can see which ones are working, their latency, and other useful information. You can also monitor the system status, view logs, and manage your settings.
