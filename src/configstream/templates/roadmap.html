Project Roadmap

Our plans for the future of ConfigStream.

Upcoming Features

  • Advanced Analytics: More detailed charts and graphs to visualize your proxy performance over time, including historical trends and provider comparisons.
  • Plugin System: An extensible plugin system to allow for custom sources, notifiers (e.g., Discord, Slack), and exporters.
  • User Authentication: Secure access to the web dashboard with user accounts and roles.
  • Improved TUI: A more feature-rich terminal user interface with more detailed information and interactive controls.

Long-Term Goals

  • Cloud Sync: Sync your settings and data across multiple devices.
  • Community-Driven Source Lists: A platform for the community to share and rate proxy sources.
  • Mobile App: A native mobile app for managing your proxy configurations on the go.

Contributions

We welcome contributions from the community! If you have an idea for a new feature or would like to help with development, please check out our GitHub repository and open an issue or pull request.
