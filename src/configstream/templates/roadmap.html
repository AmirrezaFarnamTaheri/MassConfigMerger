Project Roadmap

Our plans for the future of ConfigStream.

Upcoming Features

  • Advanced Analytics: More detailed charts and graphs to visualize your VPN performance over time, including historical trends and provider comparisons.
  • Plugin System: An extensible plugin system to allow for custom sources (e.g., other subscription formats), notifiers (e.g., Discord, Slack), and exporters (e.g., custom formats for different VPN clients).
  • User Authentication: Secure access to the web dashboard with user accounts and roles, allowing for multi-user environments.
  • Improved TUI: A more feature-rich terminal user interface for power users, with more detailed information and interactive controls.

Long-Term Goals

  • Mobile App: A native mobile app for managing your VPN configurations on the go.
  • Cloud Sync: Sync your settings and data across multiple devices using a cloud-based service.
  • Community-Driven Source Lists: A platform for the community to share and rate VPN sources.

Contributions

We welcome contributions from the community! If you have an idea for a new feature or would like to help with development, please check out our GitHub repository and open an issue or pull request.
