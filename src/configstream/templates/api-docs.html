POST /api/aggregate

Run the aggregation pipeline to fetch and process VPN configurations from configured sources.

Headers
X-API-Key string (optional) - API token if authentication is enabled
Example Request
curl -X POST http://localhost:8080/api/aggregate \\
  -H "Content-Type: application/json" \\
  -H "X-API-Key: your-token-here"
Example Response
{
  "status": "ok",
  "output_dir": "/path/to/output",
  "files": ["file1.txt", "file2.yaml"],
  "file_count": 2,
  "duration_seconds": 12.345,
  "completed_at": "2025-10-07T12:00:00Z"
}
POST /api/merge

Run the VPN merger pipeline to consolidate and test aggregated configurations.

Headers
X-API-Key string (optional) - API token if authentication is enabled
Example Request
curl -X POST http://localhost:8080/api/merge \\
  -H "Content-Type: application/json" \\
  -H "X-API-Key: your-token-here"
Example Response
{
  "status": "merge complete",
  "duration_seconds": 8.123,
  "completed_at": "2025-10-07T12:00:00Z",
  "resume_file": "/path/to/resume.txt"
}
GET /api/history

Retrieve proxy history and reliability statistics in JSON format.

Query Parameters
limit integer (optional) - Maximum number of results to return
Example Request
curl http://localhost:8080/api/history?limit=10
Example Response
{
  "items": [
    {
      "key": "proxy1.example.com:8080",
      "successes": 150,
      "failures": 10,
      "reliability_percent": 93.75,
      "status": "Healthy",
      "country": "US",
      "isp": "Example ISP"
    }
  ],
  "returned": 10,
  "total": 150,
  "healthy": 120,
  "critical": 15
}
POST /api/sources

Add a new VPN source URL to the sources list.

Request Body
url string (required) - The source URL to add
Example Request
curl -X POST http://localhost:8080/api/sources \\
  -H "Content-Type: application/json" \\
  -d '{"url": "https://example.com/subscription"}'
DELETE /api/sources

Remove a VPN source URL from the sources list.

Request Body
url string (required) - The source URL to remove
Example Request
curl -X DELETE http://localhost:8080/api/sources \\
  -H "Content-Type: application/json" \\
  -d '{"url": "https://example.com/subscription"}'
