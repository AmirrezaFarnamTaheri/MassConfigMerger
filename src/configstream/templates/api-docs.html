GET /api/current

Fetch the current, most recently tested VPN nodes.

Query Parameters:

  • protocol (string): Filter by protocol (e.g., "VMess").
  • country (string): Filter by country code (e.g., "US").
  • max_ping (integer): Filter by maximum latency.

GET /api/history

Fetch historical test data.

Query Parameters:

  • hours (integer, default: 24): How many hours of history to retrieve.

GET /api/statistics

Get aggregated statistics about the current nodes.

GET /api/export/<format>

Export the current nodes in a specified format.

URL Parameters:

  • format (string): "json" or "csv".
