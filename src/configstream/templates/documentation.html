Introduction

ConfigStream is a powerful tool for aggregating, testing, and managing VPN configurations from various sources. This documentation provides a detailed overview of its features and how to use them.

Core Features

  • Aggregation: Automatically fetches VPN configurations from multiple sources, including URLs and Telegram channels.
  • Testing: Tests the connectivity and performance of each configuration, providing metrics like latency and success rate.
  • Merging: Merges the best-performing configurations into a single, optimized subscription link.
  • Web Dashboard: A user-friendly interface for monitoring the status of your configurations, viewing test results, and managing settings.
  • API: A RESTful API for programmatic access to your configuration data.

Command-Line Interface (CLI)

The CLI is the primary way to interact with ConfigStream. Here are the main commands:

  • configstream fetch: Fetches configurations from all enabled sources.
  • configstream merge: Merges the fetched configurations into a single subscription.
  • configstream retest: Retests an existing subscription file.
  • configstream full: Runs the full fetch, merge, and test pipeline.
  • configstream daemon: Starts the web dashboard and scheduler.
