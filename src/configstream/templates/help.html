Frequently Asked Questions

What is ConfigStream?

ConfigStream is a tool for aggregating, testing, and managing VPN configurations from various sources.

How do I add new sources?

You can add new sources by editing the `sources.txt` file or by using the API.

How often are the nodes tested?

The test interval can be configured in the settings. The default is every 2 hours.
