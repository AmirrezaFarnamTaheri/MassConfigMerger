What is ConfigStream?

ConfigStream is a powerful tool for aggregating, testing, and managing VPN and proxy configurations from various sources.

How to Use the Dashboard

The main dashboard allows you to run the aggregation and merge pipelines. If an API token is configured in your config.yaml, you must provide it to run these actions.

Troubleshooting

If you encounter issues, check the Logs page for detailed error messages. Ensure your config.yaml is correctly configured and that all dependencies are installed.
